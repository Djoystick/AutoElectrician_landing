# Task 001: Complete Redesign - Auto Electrician Landing Page

## Context
You are acting as a Senior Frontend Designer and Developer. You have been tasked with completely reimagining the auto electrician landing page/business card.

## Objective
Discard the old, basic design and create a **maximally advanced, innovative, and premium** one-page website. It must look and feel like a high-end service, following the best canons of modern web development.

## Core Requirements
1. **Format**: A single-page business card (Landing Page). It should be simple in structure but visually breathtaking.
2. **Key Elements to Highlight**: The entire design must elegantly funnel the user's attention to two primary contact methods. You must integrate them beautifully and intuitively into the UI (e.g., floating action buttons, interactive glowing cards, or stylish contact blocks):
   - **Phone**: `+7 996 031 4637`
   - **Telegram**: `@DemonMSB`
3. **Niche**: Auto Electrician (Автоэлектрик). The aesthetics should subtly reflect technology, electricity, cars, and premium service without being tacky.

## Design & Aesthetic Rules (STRICT)
- **Premium Aesthetics**: Use modern UI trends. We expect glassmorphism, subtle glowing neon accents (like electric blue or electric green on a deep dark theme), or sleek minimal dark mode.
- **Micro-animations**: Elements must feel alive. Add hover effects, smooth entrances, and pulse animations for the contact buttons. Use `framer-motion` or advanced Tailwind animations if applicable.
- **Typography**: Use modern, clean fonts (e.g., Inter, Roboto, or Outfit).
- **No generic colors**: Use curated, rich color palettes (e.g., Slate-900 background with Cyan-400 glowing accents).

## Output
Review the current codebase, wipe the outdated styles, and implement your new vision. When finished, create a short report in `.mimo_workflow/001_report.md` explaining your design choices.
