# Отчёт: Задание 006 — Layout Rebalance (Hero & Services)

## Дата: 2025-06-11

## Проблема
- Hero был пустым — чёрный void без содержимого
- Services были перенасыщенными — гигантские карточки, длинный заголовок

## Решения

### Hero — центрированный и ударный
- **Центрирование**: `flex flex-col items-center text-center` + `justify-content: center`
- **Заголовок**: "УСЛУГИ АВТОЭЛЕКТРИКА" с electric shock — главный H1 в самом верху
- **CTA**: "Связаться с мастером" — сразу виден
- **Статистика**: по центру, компактная

### Services — компактные и читаемые
- **Новый заголовок**: "Мои услуги" (вместо дублирования "Услуги автоэлектрика")
- **Уменьшенный section-title**: `clamp(28px, 4vw, 48px)` вместо `clamp(36px, 5vw, 64px)`
- **Убран `margin-bottom: 56px`** у section-title
- **Карточки компактнее**:
  - Padding: `24px` вместо `36px`
  - Min-height: `200px` вместо `260px`
  - Иконка: `40px` вместо `52px`
  - Шрифт заголовка: `14px` вместо `16px`
  - Шрифт описания: `13px` вместо `14px`
  - Bottom padding: `14px` вместо `20px`
- **Описания укорочены**: каждый сервис — 1-2 предложения
- **Мобильные**: padding `20px`, min-height `170px`

## Изменённые файлы

| Файл | Изменения |
|------|-----------|
| `src/components/Hero.jsx` | Центрирование flex-col + items-center + text-center |
| `src/components/Services.jsx` | Новый заголовок "Мои услуги", компактные описания |
| `src/index.css` | Компактные карточки, уменьшенные размеры, центрированный hero |
