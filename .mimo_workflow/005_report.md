# Отчёт: Задание 005 — Hero Restructure & Services Upgrade

## Дата: 2025-06-11

## Проблема
- Первый экран был пустым, заголовок тонул ниже fold, гигантский SVG-болт не нужен
- Бейдж "Профессиональный автоэлектрик" — мусор
- Заголовок "АВТО M ЭЛЕКТРИК" в навбаре — сухой
- Карточки услуг — мёртвые, плоские, без жизни

## Решения

### Hero — компактный и плотный
- **Удалено**: SVG-графика, бейдж, giant lightning bolt
- **Главный H1**: "УСЛУГИ АВТОЭЛЕКТРИКА" с electric shock эффектом — теперь самый верх экрана
- **Продающий текст**: точная копия из ТЗ — "Быстро. Точно. Надёжно..."
- **CTA**: "Связаться с мастером" — яркая кнопка, сразу видна без скролла
- **Layout**: max-width 5xl, компактные отступы, нет пустых пространств

### Navbar — без сухого заголовка
- Убрано "АВТО M ЭЛЕКТРИК"
- Теперь: M-бейдж + "ЭЛЕКТРИК" — лаконичнее

### Services — живые карточки
- **Новый CSS**: `.service-card` вместо `.card`
- **M-tricolor полоса**: анимируется от 0 до 100% ширины при hover
- **Подъём**: `translateY(-4px)` + `box-shadow` при наведении
- **Иконка пульсирует**: `@keyframes iconPulse` — scale 1 → 1.08
- **"Подробнее" появляется**: opacity 0 → 1 с подъёмом
- **Стрелка сдвигается**: gap 6px → 10px при hover
- **Типографика**: шрифт описаний увеличен до 14px, контраст улучшен
- **Сетка**: 3 колонки десктоп, 2 планшет, 1 мобайл

## Изменённые файлы

| Файл | Изменения |
|------|-----------|
| `src/components/Hero.jsx` | Полная переработка: компактный layout, shock-text H1, новый CTA |

| `src/components/Navbar.jsx` | Убран заголовок, новый nav-logo-link с бейджем |
| `src/index.css` | Новые стили service-card, компактный hero, удалены SVG-стили |
